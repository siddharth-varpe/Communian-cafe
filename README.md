## 🌟 Overview

The Communion Cafe website is a performant, high-converting React application designed to represent the cafe's premium yet welcoming brand aesthetic. The application is built to prioritize user conversion: driving inbound phone calls, facilitating online table reservations, increasing online order clicks, and showcasing the beautiful "Google Review" social proof of the actual physical location.

## ✨ Key Features

- **High-Converting Hero Section**: Immediate calls-to-action ("Call Now", "Reserve a Table") highlighting Google Review stats.
- **Dynamic Signature Dishes Grid**: Curated popular vegetarian items featuring high-quality photography.
- **Custom Cursor Trail**: A lightweight, performant HTML5 Canvas-based pixel trail effect that perfectly matches the cafe’s exact color palette.
- **"Loved By Guests" Social Proof**: A dedicated section displaying verified Google Reviews and a mock Instagram feed.
- **Smooth Reservation Form**: Clean, modern, and accessible input fields for fast bookings.
- **Floating WhatsApp Integration**: Persistent floating action button that enables immediate, friction-free WhatsApp inquiries with pre-filled messages.
- **Fully Responsive Architecture**: A 100% responsive, mobile-first design guaranteeing flawless viewing across all viewports (from 320px mobiles to 1440px+ ultra-wide displays).
- **SEO Optimized**: Pre-configured with meta tags, descriptions, and `LocalBusiness` JSON-LD schema markup optimized for local Charholi / Lohegaon search traffic.

## 🛠️ Tech Stack & Technologies Used

- **Framework**: [React 18](https://reactjs.org/)
- **Build Tool**: [Vite](https://vitejs.dev/) (For blazing-fast Hot Module Replacement and highly optimized production bundling)
- **Styling**: Vanilla CSS (Modular, scalable, and responsive without relying on heavy frontend frameworks)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Animations**: CSS Keyframe animations & Native Canvas API (Custom Cursor Trail)

## ⚙️ Running Locally

Follow the steps below to run this project in your local development environment:

### Prerequisites

You will need **Node.js** (v16.0 or higher recommended) and **npm** installed on your machine.

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/siddharth-varpe/communian-cafe.git
   ```

2. Navigate into the project directory:
   ```bash
   cd communian-cafe
   ```

3. Install the dependencies:
   ```bash
   npm install
   ```

4. Start the local Vite development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:5173](http://localhost:5173) to view it in the browser!

## 📦 Building for Production

To create an optimized production build:

```bash
npm run build
```
The output will be generated inside the `/dist` directory, which can act as a static site and be hosted freely on platforms like Vercel, Netlify, Hostinger, or GitHub Pages!
